# Исследование продаж компьютерных игр

## Данные
Для целей данного исследования в распоряжении есть данные о количестве проданных копий компьютерных игр в разбивке по регионам: Северная Америка, Европа, Япония и другие, а также несколько видов рейтингов игры: рейтинг критиков (максимальный балл 100), рейтинг пользователей (максимум 10), и рейтинг от компании ESRB (Entertainment Software Rating Board).
- название игры
- платформа
- год выхода 
- жанр
- продажи в Северной Америке
- продажи в Европе
- продажи в Японии
- продажи в других регионах
- оценка критиков
- оценка пользователей
- рейтинг 

## Задача
Изучить данные о продажах игр из открытых источников; выявить закономерности, определяющие успешность игры, что позволит в дальнейшем определить потенциально популярный продукт и спланировать рекламную активность.
Проверить  ряд гипотез.

## Используемые библиотеки, инструменты и навыки
*pandas, matplotlib, seaborn, SciPy* , исследовательский анализ данных, предобработка данных, визуализация, t-критерий Стьюдента.
